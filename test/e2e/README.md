# E2E test suite

Vitest tests that drive the running app via Playwright. They replace
the older `scripts/verify*.mjs` / `scripts/probe*.mjs` files (which
mixed test logic with raw `process.exit`).

## Running

```bash
pnpm dev          # in one terminal — leave running
pnpm test         # in another
```

The suite assumes the dev server is already up at
`http://localhost:5173`. Tests don't spawn or kill the server.

## Fixtures

* `test/fixtures/maldivian.pdf` — the canonical Maldivian government
  doc with the broken-aabaafili `/ToUnicode` CMap. Pinned input for
  the Thaana-recovery + edit/move suite.
* `test/fixtures/with-images.pdf` — a synthetic A4 page with two
  known-position PNG images, generated by
  `test/fixtures/build.mjs`. Used by the image-move + preview-strip
  + insert tests so we don't pin against any user-specific PDF.

Regenerate the synthetic fixture with:

```bash
node test/fixtures/build.mjs
```

## Files

| File                          | What it covers                                |
| ----------------------------- | --------------------------------------------- |
| `move-edit.test.ts`           | move-only / edit-only / move+edit on Maldivian |
| `image-move.test.ts`          | drag image → cm rewrite, neighbours unchanged |
| `preview-strip.test.ts`       | original glyph removed from canvas (no cover) |
| `insert.test.ts`              | drop text + image → both persist after save   |
| `insert-format.test.ts`       | font / size / bold round-trip from toolbar    |
